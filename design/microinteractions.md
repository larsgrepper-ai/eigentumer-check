# Microinteraction Plan

1. **Tool-Karten (Landing)**
   - Hover: leichtes Scale + Shine-Overlay.
   - Click: Button ripple, Laden-Skeleton → führt zu Tool.

2. **CTA-Buttons**
   - Add subtle gradient shift + box-shadow pulse alle 8s.

3. **Stat-Kacheln**
   - Count-up Animation beim Scroll (IntersectionObserver).

4. **Formularfeedback**
   - Inline Validation + Success Toast (Mailto später ersetzen durch webhook).

5. **Timeline** (künftige Silo-Sections)
   - Stepper mit progress line, mobile swipeable.
