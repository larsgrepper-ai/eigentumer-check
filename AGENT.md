# Projekt
SEO-Portal für Schweizer Immobilieneigentümer in Entscheidungssituationen.

# Branch-Regeln
- staging = Testumgebung
- main = Live-Umgebung

# Deploy-Regeln
- Push auf staging deployt auf staging.eigentuemer-check.ch
- Push auf main deployt auf eigentuemer-check.ch

# Struktur
- index.html = Startseite
- /tools/<tool-name>/index.html = Tools
- /silos/<thema>/index.html = SEO-Silo-Seiten
- assets = CSS, JS, Bilder
- data = Content- und Designregeln

# Schreibstil
- seriös
- schweizerisch
- präzise
- keine generischen KI-Floskeln
- keine übertriebenen Versprechen

# Jede Seite braucht
- Meta Title
- Meta Description
- H1
- Einleitung
- Nutzenblock
- CTA
- FAQ
- interne Links
- Kontaktmöglichkeit

# Verboten
- US-Marketing-Ton
- falsche Schweizer Rechtsbehauptungen
- leere Fülltexte
- chaotische Dateistrukturen

# Tool-Regeln
- jedes Tool bekommt einen eigenen Ordner unter /tools/
- jede Tool-Seite muss eigenständig funktionieren
- keine zerstörerischen Änderungen an bestehenden Seiten ohne klaren Grund
