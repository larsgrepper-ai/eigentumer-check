# Komponentenbibliothek – Homepage

## Struktur
- `home/hero.html` – Navigation, Hero, Metrics
- `home/kpis.html` – KPI-Strip
- `home/tools-grid.html` – Wrapper für Rechner (siehe `index.html` für Inputs)
- `home/values.html` – Werte-Sektion

## Nutzung
1. Beim Erstellen neuer Landingpages die benötigten Komponenten per Include oder Copy verwenden.
2. Styles liegen in `index.html` (künftig nach `assets/css/main.css`).
3. Änderungen immer gleichzeitig in der Komponente und im Haupttemplate pflegen.
